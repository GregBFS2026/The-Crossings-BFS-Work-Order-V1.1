BFS Work Order Web App Prototype

1. Unzip the folder.
2. Open index.html in a browser to test it on a computer.
3. For phone testing, upload the folder to a simple web host such as Netlify, GitHub Pages,
   or your company website.
4. The Submit button currently opens the user's email app and prepares an email to:
   gregcarrillo@bfsflooring.com
5. Photo files cannot be attached automatically through a mailto link. A production version
   will require a secure backend or form service to save submissions and photos automatically.

Updated: Each installation location now has a dropdown with None, LVT, Carpet, or Both.
